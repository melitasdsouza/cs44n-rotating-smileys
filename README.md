# Rotating Smiley Grid

Generative animation created for **CS 44N: Great Ideas in Graphics** at Stanford.

A grid of smiley faces rotates continuously, with alternating rotation directions and hue-shifted colors driven by frame count.

## Live Demo
- OpenProcessing: oppr.org/s/p772WnTp

## Features
- p5.js animation
- HSB color mode with dynamic hue cycling
- Alternating rotational symmetry
- Modular drawing functions

## Technologies
- JavaScript
- p5.js

## Notes
This repository contains the standalone version of the sketch, adapted from OpenProcessing for archival and portfolio use.
